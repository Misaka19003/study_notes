# 量子编程
*以下教程将在Windows-python-cmd/powershell环境下运行。*
## Qiskit
Qiskit是IBM开发的开源量子计算SDK。
### 初始化教程
Qiskit是基于python语言开发的，请确保电脑上正确安装了python 3.10及以上的版本。
#### Qiskit虚拟环境
创建Qiskit虚拟环境，防止库或环境冲突。
```
python -m venv .venv
```
激活环境。
```
.venv\Scripts\activate
```
#### 安装 Qiskit以及其组件
```
pip install qiskit[visualization] qiskit-aer matplotlib
```
其中qiskit是核心库以及其可视化组件[visualization]、qiskit-aer是量子模拟器、matplotlib是图形化工具。
### 检查安装
@import "./codes/检查安装.py"
若输出版本号，例如
```
2.2.3
```
则表示成功安装qiskit。
