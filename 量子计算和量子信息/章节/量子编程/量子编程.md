# 量子编程
*以下教程将在Windows-python-cmd/powershell环境下运行。*
## Qiskit
Qiskit是IBM开发的开源量子计算SDK。
### 初始化教程
Qiskit是基于python语言开发的，请确保电脑上正确安装了python 3.10及以上的版本。
#### Qiskit虚拟环境
创建Qiskit虚拟环境，防止撞库或者
```
python -m venv .venv
```
```
.venv\Scripts\activate
```
#### 安装 Qiskit以及其组件
```
pip install qiskit qiskit-aer matplotlib
```
其中qiskit是核心库、qiskit-aer是量子模拟器、matplotlib是图形化输出工具。
### 检查安装
```
import qiskit
print(qiskit.__version__)
```
