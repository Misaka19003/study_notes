# 量子编程
## Qiskit
以下使用Qiskit的教程将在Windows-python-cmd/powershell环境下运行
### 初始化教程

#### 安装 Qiskit以及其组件
```
pip install qiskit qiskit-aer matplotlib
```
其中qiskit是核心库、qiskit-aer是量子模拟器、matplotlib是图形化输出工具。
### 检查安装
```
import qiskit
print(qiskit.__version__)
```
#### Qiskit 虚拟环境
```
python -m venv .venv
```
```
.venv\Scripts\activate
```
